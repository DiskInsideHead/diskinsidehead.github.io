DmElement to QC Converter

Convert Source Engine DmElement facial animation data to QC format

Input DmElement

Output QC

How to Use

  • Replace all "_" or " " with "|" in shapekeys of your mesh in 3D software
  • Generate and paste your DmElement facial animation data in the left panel
  • Click "Convert" to generate QC code
  • Copy the output to your Source Engine QC file
  • Replace "[FACE]" with your model name and "head.dmx" with your mesh name
